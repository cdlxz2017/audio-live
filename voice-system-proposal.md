## 五、技术选型对比

## 六、数据库设计

```sql
-- 通话记录
CREATE TABLE calls (
    id SERIAL PRIMARY KEY,
    caller VARCHAR(20),        -- 主叫号码
    callee VARCHAR(20),       -- 被叫号码
    direction VARCHAR(10),     -- 'inbound' / 'outbound'
    status VARCHAR(20),        -- 'answered' / 'missed' / 'voicemail'
    duration INT,              -- 通话秒数
    recording_path TEXT,       -- 录音文件路径
    transcript TEXT,           -- Whisper转写
    summary TEXT,              -- LLM摘要
    emotion VARCHAR(20),       -- 情感标签
    tags TEXT[],               -- 自定义标签
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 短信记录
CREATE TABLE sms (
    id SERIAL PRIMARY KEY,
    from_number VARCHAR(20),
    to_number VARCHAR(20),
    direction VARCHAR(10),
    content TEXT,
    status VARCHAR(20),
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 联系人
CREATE TABLE contacts (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    phone VARCHAR(20) UNIQUE,
    remark TEXT,
    is_blacklist BOOLEAN DEFAULT FALSE,
    is_whitelist BOOLEAN DEFAULT FALSE,
    tags TEXT[],
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 定时任务
CREATE TABLE schedule_tasks (
    id SERIAL PRIMARY KEY,
    task_type VARCHAR(20),    -- 'call' / 'sms'
    target VARCHAR(20),        -- 目标号码
    content TEXT,              -- 拨打时说的内容 / 短信内容
    scheduled_at TIMESTAMPTZ,
    status VARCHAR(20),        -- 'pending' / 'done' / 'failed'
    result TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

## 八、第一步行动清单（4G模块到手后）

1. `ls /dev/ttyUSB*` — 确认模块识别
2. `sudo apt install modemmanager modemmanager-bin mmcli` — 安装ModemManager
3. `mmcli -L` — 查看模块
4. `sudo apt install freeswitch` — 安装FreeSWITCH
5. 配置FreeSWITCH SIP分机
6. linphone 注册测试
