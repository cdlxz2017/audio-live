# Hermes Router

## 基本信息
- **类型**：路由工具
- **路径**：`custom-skills/hermes-router/`
- **来源**：自制
- **状态**：✅ 正常

## 能力
- 自动判断任务复杂度，决定是否需要委托给 Hermes Agent
- 复杂度 0-10 评分

## 路由规则
| 复杂度 | 行动 |
|--------|------|
| 0-3 | OpenClaw 直接处理 |
| 4-6 | 根据任务时长判断 |
| 7-10 | 委托给 Hermes Agent |

## 委托场景
- 代码分析/设计/架构/重构/调试
- 研究调研/比较/评估
- 深度研究/调研报告
- 关键词：分析、比较、设计、研究、评估、优化、架构、why、how

## 直接处理场景
- 简单查询：什么/谁/什么时候/在哪里
- 记忆存取：记住/查一下/告诉我
- 日常对话：问候/闲聊/天气
- 任务管理：增删改查

## 调用
```bash
cd /home/ai/apps/hermes-agent && python3 -m hermes_cli --prompt "你的问题" --no-interactive
```
