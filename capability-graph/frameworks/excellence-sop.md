# 卓越执行框架

## 基本信息
- **类型**：SOP / 任务执行标准流程
- **版本**：v2.1
- **日期**：2026-04-19
- **路径**：capability-graph/SOP-EXCELLENCE-FRAMEWORK.md

## 核心原则
1. 已知问题 → 快速复用历史成功方案
2. 未知问题 → 谨慎探索，多方案对比
3. 危险点前置 → 动手之前先找危险，解决后再干
4. 文档闭环 → 做完必更新，为下次复用做准备

## 任务分级
| 级别 | 条件 | 通道 |
|------|------|------|
| Trivial | 总分≥10 且 可逆≥2 且 确定性=3 | 快速通道 |
| Standard | 总分≥7 且 确定性≥2 | 标准通道 |
| Complex | 总分<7 或 确定性≤1 | 完整通道 |
| Dangerous | 可逆=0 或 影响范围=0 | 完整通道 + 呈报 |

## 三级通道
- 🔵 快速：执行→验证→完成（30秒）
- 🟢 标准：检索→检查→扫描→确认→执行→验证→记录（5分钟）
- 🔴 完整：问题定义→已知/未知分流→多方案→确认→危险识别→工具查阅→执行→验证→文档闭环

## 团队模式
- 触发：Complex/Dangerous 任务满足特定条件
- 实时探测 → 动态组队 → 互查机制 → 合并交付
