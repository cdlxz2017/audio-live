# 摄像头录制系统

## 基本信息
- **类型**：硬件控制
- **路径**：`custom-skills/camera-recorder/`
- **来源**：自制
- **状态**：✅ 正常

## 能力
- 打开摄像头（预览不录制）
- 录制视频+音频（H.265 压缩）
- 语音转写（Whisper）
- 自动邮件发送视频
- 3分钟静止自动停止

## 配置
| 项 | 值 |
|----|-----|
| 视频目录 | ~/videos |
| 编码 | H.265, CRF 28 |
| 转写 | Whisper large-v3-turbo |
| 数据库表 | video_records（独立表） |

## 命令
| 命令 | 功能 |
|------|------|
| 打开摄像头 | 预览流，不录制 |
| 开始录制 | 开始录制视频+音频 |
| 停止录制 | 保存+转写+发邮件 |
| 录制状态 | 查看当前状态 |

## 依赖
- ffmpeg (libx265)
- Whisper CLI
- Python 3
